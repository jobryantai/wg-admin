# WireGuard Admin Panel — Phase One Project Files (GitHub + Gitea)

This ZIP contains all documentation and planning assets for Phase One of the WireGuard Admin Panel project, updated to support:

- GitHub as the primary remote  
- Gitea as a mirrored DevOps environment  
- Dual‑remote workflow  
- Deployment pulling from GitHub  

## Contents
- `PHASE_ONE_PLAN.md` — Full project plan  
- `TASKS.csv` — Importable task list for Gitea  
- `docs/ARCHITECTURE_OVERVIEW.md`  
- `docs/DATABASE_SCHEMA.md`  
- `docs/DEPLOYMENT_WORKFLOW.md`  
- `docs/SECURITY_MODEL.md`  
- `docs/GITHUB_GITEA_WORKFLOW.md`  

## Usage
1. Import `TASKS.csv` into Gitea  
2. Use GitHub as your primary remote  
3. Let Gitea mirror GitHub automatically  
4. Use `PHASE_ONE_PLAN.md` as your execution guide
